### WeEvolve -- Self-Evolving Agent Protocol

- **Install:** `pip install weevolve` or `npx skills add 8owls/weevolve`
- **What:** The SEED protocol teaches your agent to learn how to learn. Every input runs through 8 recursive phases (PERCEIVE, CONNECT, LEARN, QUESTION, EXPAND, SHARE, RECEIVE, IMPROVE). MMORPG progression tracks growth across 14 skill categories. Genesis knowledge transfer lets agents bootstrap from shared intelligence. Auto-trigger hooks activate on every interaction. 8 Owls emergence provides multi-perspective analysis. Persistent memory compounds across sessions.
- **Platforms:** Claude Code, OpenClaw, Cursor, Codex
- **GitHub:** [aro-brez/weevolve](https://github.com/aro-brez/weevolve)
- **Website:** [8owl.ai](https://8owl.ai)
